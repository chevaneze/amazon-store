# amazon-store

This is a command-line node app that uses mySQL to store and enter data in the form of an online retailer.

To get started, simply input "node bamazon.js" in the terminal and follow the prompt!

Demo: https://www.youtube.com/watch?v=BGaVlwVziMw&feature=youtu.be

Github Repository: https://github.com/chevaneze/amazon-store
